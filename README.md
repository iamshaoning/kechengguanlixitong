## 功能特性

### 核心功能

- **日历视图**：直观的月历展示，快捷添加课程，查看课程详情，支持修改、复制、粘贴
- **快速导航**：支持通过箭头快速切换月份，年月下拉菜单便于快速跳转
- **学生管理**：添加、编辑、删除学生信息，可预设一对一课时费，机构管理，年级管理
- **课程管理**：支持一对一、一对多课程安排，课时费可按整节课自定义，也可加载预设根据时长自动计算
- **数据统计**：按月、按机构统计课程和收入。支持按机构课型年级学生人数详细展示课节分布，支持导出直观的数据

### 数据管理

- **机构管理**：自定义机构名称，支持增删改
- **年级管理**：自定义年级名称，支持增删改
- **数据同步**：本地数据自动同步到云端，跨设备同步，跟账号绑定
- **数据冲突处理**：智能比较本地和服务器数据，支持手动选择

### 用户体验优化

- **全局屏蔽右键菜单**：防止意外触发浏览器菜单
- **隐藏滚动条**：保持界面整洁，同时保留滚动功能
- **响应式设计**：适配不同屏幕尺寸
- **流畅动画**：页面切换、模态框等交互采用流畅动画效果

## 技术特点

- **开源字体**：使用 LXGW WenKai（霞鹜文楷）开源中文字体
- **现代前端技术**：基于 Vite + Tailwind CSS 构建
- **数据可视化**：集成 Chart.js 实现数据图表展示
- **云端同步**：基于 Supabase 实现数据云端存储

## 使用指南

### 登录/注册

- 首次使用需要注册账号
- 支持邮箱密码登录
- 登录后可自动同步数据到云端

### 添加年级和机构数据

- 先添加年级和机构数据

### 添加学生

1. 进入"学生管理"页面
2. 点击"添加学生"按钮
3. 填写姓名、选择机构和年级
4. 设置课型费用
5. 保存

### 安排课程

1. 在日历视图中点击日期选择添加
2. 选择学生、时间、时长
3. 保存课程

### 数据统计

1. 进入"统计"页面
2. 选择年份、月份、机构
3. 查看课程数量、收入统计
4. 导出HTML文件报表

## 数据存储

- 登录后自动同步
- 支持多设备同步

## 注意事项

1. **网络要求**：登录加载和同步都需要网络连接
2. **浏览器支持**：建议使用现代浏览器获得最佳体验
